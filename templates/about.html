About CyberGuard

CyberGuard is an open-source cybersecurity toolkit designed for developers, security researchers, and anyone who wants to strengthen their digital security posture.

Modules

  • ▸ Password Checker — Evaluate password strength using entropy, length, and character diversity analysis.
  • ▸ Password Generator — Create secure random passwords with configurable length and character sets.
  • ▸ Hash Generator — Compute cryptographic hashes using MD5, SHA-1, SHA-256, and SHA-512 algorithms.
  • ▸ Port Scanner — Discover open ports on target hosts to identify accessible services.
  • ▸ Phishing Detector — Analyze URLs for suspicious patterns, typosquatting, and known phishing indicators.

Tech Stack

Built with Python, Flask, Jinja2, and Tailwind CSS. Designed with a sleek dark cybersecurity aesthetic.

Disclaimer

This tool is for educational and authorized security testing purposes only. Always obtain proper authorization before scanning networks or testing systems you do not own.
